# Pedro-Eugelmi | Marketing Landing Page for SENNA

A custom, high-performance WordPress theme originally developed as a college project. This theme is built as a conversion-focused Landing Page, featuring modular PHP architecture and automated asset optimization.

## 🚀 Features

* **ACF Integration:** Includes `acf-json` for automated synchronization of Advanced Custom Fields across environments.
* **Gulp Workflow:** Automated task runner for compiling styles, minifying scripts, and optimizing assets.
* **Dynamic Templates:** Custom page templates for `Home` and `Blog` layouts.
* **Performance First:** Serves minified JS and CSS from the `/minified` directory to ensure high Core Web Vitals scores.
